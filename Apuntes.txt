PARA CORRER EN EL EQUIPO: 
* npm run dev // Compila CSS y Javascript
* npm install (una sola vez) // Instalar dependencias de Node
* composer require laravel/breeze --dev // Instalar Breeze con TailwindCSS
* php artisan breeze:install // Instalar sistema de autenticacion (escoger opcion blade) (el que crea el proyecto lo ejecut, los demas no tienen la necesidad de hacerlo)
* php artisan serve
